# AGENTS.md - Exportacao De Dados

Regras obrigatorias:

1. Toda exportacao deve respeitar company_id.
2. Nunca exportar dados de outra empresa.
3. CSV deve ser compativel com Excel brasileiro.
4. Usar UTF-8 com BOM quando necessario.
5. Valores monetarios devem ser exportados de forma consistente.
6. Datas devem ter formato claro.
7. Aplicar os mesmos filtros usados na listagem.
8. Nao criar exportacoes avancadas fora do escopo.
9. Criar testes de autorizacao e filtros.
